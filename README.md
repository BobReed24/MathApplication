# MathApplication

## Requirements
 - Java Development Kit (JDK) 17 (61.0) or higher
 - Gradle 8.12.1 (recommended version)

## How to use
 - To build run
   ```ruby
   gradle clean build
   ```
- To start run
  ```ruby
  gradle run
  ```
- If the first run command does not work then try
 ```bash
 /path/to/your/java17/bin/java -jar app/build/libs/app-all.jar
 ```
